💡 为什么显性层板用14天窗口？
显性层板是功能性购买，买家通常有明确需求（如装修、搬家），决策周期短。超过14天未购买的用户，需求可能已被满足或放弃，继续投放ROI较低。
📢 广告投放策略
  • 投放广告类型：展示型推广（SD），高竞价优先抢占
  • 素材重点：强调承重数字 + 安装简便（5–10分钟）+ 36"/48"加L型角码
  • 加入社会证明：评分/评论数量（如有）
  • 可配合优惠券或限时折扣，加速决策

SparkX 后台操作步骤
在 Xnurta/SparkX 平台创建上述受众的完整操作流程
⚠️ 前置条件
登录 SparkX 后，必须先切换到 KovaScape北美（AMC类型） 账户，AMC Instance ID：amcump7aucq。默认进入的是"南京欧洲组KS"SP账户，需手动切换。
1
登录并切换账户
访问 ai.sparkx.io → 右上角第2个下拉框 → 选择"北美洲" → 选择"KovaScape北美（AMC）" → 确认URL中包含 instanceId=21834
2
进入受众创建页面
左侧导航 → AMC → 受众管理 → 点击"创建受众"按钮 → 选择受众模板（规则型受众 或 相似受众）
3
填写受众人群区块
名称：按命名规范填写（如 KS_显性层板_点击未购_14D_20260519）
目标广告账户：选择 KovaScape 北美
自动更新：开启，频率按方案设置（1天/3天/7天/14天）
受众类型：规则型受众 或 相似受众
邻近度级别：相似受众选1级或1–2级
4
配置受众条件
时间设置：选择"为每个条件组"（单条件）或"为所有条件组"（多条件AND）
基础受众：选择"被曝光" / "已点击" / "转化事件"
转化事件：加入购物车 / 购买 / 商品详情页浏览
数值类型：不重复数量（统计独立用户行为次数）
运算符+阈值：≥ 1 / ≥ 3 / ≥ $30（按方案设置）
5
设置排除条件（重要）
点击"添加排除条件" → 选择"转化事件 → 购买" → 时间窗口与主条件相同
目的：排除已购买用户，避免对已转化用户浪费预算，同时避免受众重叠
6
保存并关联广告活动
保存受众 → 等待受众规模计算（通常24–48小时）
确认规模 ≥ 2,000人（Sponsored Ads最低要求）
在广告活动中选择"受众定向" → 选择已创建的AMC受众 → 设置竞价
