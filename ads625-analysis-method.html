一
核心框架：625矩阵
4个核心指标 × 5个等级 = 5⁴ = 625种组合
矩阵原理

每个广告活动/关键词，通过4个维度的等级判断，定位到625种状态之一，并匹配对应的操作策略。

四大核心指标

指标英文含义判断维度
ACOSAdvertising Cost of Sales广告花费 / 广告销售额盈利能力
曝光量Impressions广告展示次数流量规模
CTRClick-Through Rate点击率 = 点击 / 曝光广告吸引力
CVRConversion Rate转化率 = 订单 / 点击Listing转化力

五级评分标准

ACOS等级（越低越好）
  • L1  ACOS < BE_ACOS × 60%（极优，大幅盈利）
  • L2  ACOS < BE_ACOS × 80%（优，盈利充裕）
  • L3  ACOS ≈ BE_ACOS（平衡，保本）
  • L4  ACOS < BE_ACOS × 130%（差，轻微亏损）
  • L5  ACOS > BE_ACOS × 130%（极差，严重亏损）
曝光量等级（越高越好）
  • L1  曝光量极高（类目头部水平）
  • L2  曝光量高（稳定流量）
  • L3  曝光量中等（有增长空间）
  • L4  曝光量低（流量不足）
  • L5  曝光量极低（<100/周）
CTR等级（行业基准 0.3-0.5%）
  • L1  CTR > 0.8%（极高）
  • L2  CTR 0.5-0.8%（高）
  • L3  CTR 0.3-0.5%（中等，行业平均）
  • L4  CTR 0.15-0.3%（低，需优化主图）
  • L5  CTR < 0.15%（极低，严重问题）
CVR等级（行业基准 5-15%）
  • L1  CVR > 15%（极高）
  • L2  CVR 10-15%（高，转化良好）
  • L3  CVR 5-10%（中等，行业平均）
  • L4  CVR 2-5%（低，Listing需优化）
  • L5  CVR < 2%（极低，严重问题）
二
核心公式体系
ACOS / BE_ACOS / Bid / TACoS 完整公式
基础公式
ACOS     = 广告花费 / 广告销售额 × 100%
         = CPC / (售价 × CVR)

BE_ACOS  = 毛利润 / 售价
         = (售价 - 成本 - FBA费用 - 平台佣金) / 售价

目标出价  = 目标ACOS × 售价 × CVR

TACoS    = 总广告花费 / 总销售额（含自然单）

三
铁律三条（Golden Rules）
违反任何一条都会导致错误决策
⚠️ 铁律一：CTR低 ≠ 广告问题

CTR低的根本原因：主图质量差、价格竞争力不足、评论数量/评分不足、关键词相关性差。
正确应对：先优化主图/价格/评论，再调整广告出价。不要因为CTR低就盲目降低出价。

⚠️ 铁律二：CVR低 ≠ 广告问题

CVR低的根本原因：Listing质量差、价格偏高、评论质量差、主图/副图不够专业。
正确应对：先优化Listing/价格/评论，再考虑广告策略调整。降低出价不能解决CVR问题。

💡 铁律三：ACOS不是越低越好

ACOS < BE_ACOS = 盈利，可继续投放  |  ACOS = BE_ACOS = 保本，需优化  |  ACOS > BE_ACOS = 亏损，需控制

ACOS过低（<BE_ACOS×50%）= 可能错失流量机会，应考虑扩量。
正确目标：在盈利前提下最大化销售额，而非单纯追求低ACOS。

下载上周广告数据 → 检查ACOS/CTR/CVR/花费/销售额 → 寻找±20%异常波动 → 标记问题项 → 计算TACoS对比月度目标

否词优化

新品期 vs 成熟期策略差异

🌱 新品期（0-90天，评论<50条）
  • 目标：积累数据、获取评论、建立排名
  • ACOS容忍度：BE_ACOS × 150-200%
  • 出价：计算值基础上 +20-30%
  • 竞价策略：动态（提高和降低）
  • 广告位：优先TOS，溢价20-50%
  • 否词标准：放宽至点击≥10+0转化
🌳 成熟期（90天+，评论>50条）
  • 目标：盈利最大化，降低广告依赖
  • ACOS目标：< BE_ACOS × 80%
  • TACoS目标：< 10-15%
  • 自然订单占比目标：> 60%
  • 广告订单占比目标：< 40%
  • 否词标准：严格执行（点击≥5+0转化+花费>$5）

TACoS监控维度

TACoS区间状态解读
< 10%极优广告效率极高，自然流量强
10-15%健康健康区间，广告与自然平衡
15-25%需优化广告依赖偏高，需要优化
> 25%警戒广告严重亏损或自然流量极弱

TACoS趋势组合解读

  • TACoS↓ + ACOS稳定 → 自然流量在增长（好信号）
  • TACoS↑ + ACOS稳定 → 广告花费增加但自然未跟上（需观察）
  • TACoS↑ + ACOS↑ → 广告效率下降（需优化）
  • TACoS↓ + ACOS↑ → 广告亏损但自然流量强（可接受）

常见误区

❌ 误区✅ 正确做法
CTR低就降出价先优化主图/价格/评论
CVR低就暂停广告先优化Listing/价格/评论
ACOS越低越好ACOS<BE_ACOS即可，过低应扩量
新品期追求低ACOS新品期容忍高ACOS，优先积累数据
否词越多越好否词过多会限制流量，需平衡
广告活动越多越好精简活动，集中预算在优质词上

核心公式速查

公式速查表
ACOS     = 广告花费 / 广告销售额
BE_ACOS  = 毛利润 / 售价
目标出价  = 目标ACOS × 售价 × CVR
TACoS    = 总广告花费 / 总销售额（含自然单）
CTR      = 点击量 / 曝光量
CVR      = 订单量 / 点击量
CPC      = 广告花费 / 点击量
CPA      = 广告花费 / 订单量
ROI      = 广告销售额 / 广告花费
